Cursor v46.3 is out! In this release we have new features and improvements. Agent is now the default chat mode instead
of the confusing Chat, Composer, and Agent/Normal modes. Along side this there are two new AI models available:
claude-3.7-sonnet and claude-3.7-sonnet-thinking. My short time testing with these new models so far has been positive.
Models can now also search the web by default without needing to specify the `@web` tag.
Another change with this update is some new UI themese. I clicked through a few of these and they are nice.

You can read the [changelog](https://www.cursor.com/changelog) for more details.
